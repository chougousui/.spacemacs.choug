* Description
一个单纯使用fcitx.el包的layer,帮助用户智能地,及时地切换输入法

** Features:
  - 智能切到英语状态,智能切回

* Install
To use this configuration layer, add it to your =~/.spacemacs=. You will need to
add =fcitx= to the existing =dotspacemacs-configuration-layers= list in this
file.
